Intro & Navigation

The following components are used for an introductory statement on the landing page, external social media navigation, and the site's navigation bar.

Intro & Social Bar

For Desktop/Tablet

This is the component used for the landing page welcoming intro and social media bar.


For Mobile

This is the component used for the landing page welcoming intro and social media bar. It is a condensed version of the desktop/tablet component that also eliminates the decorative bar.


Usage & Best Practices

These components should only be used on a landing homepage. They serve as a welcome and introduction to visitors of the site. The components are displayed using a dark (#202022) background as this is how they are used in the original portfolio site. Best practice would be to use them on a dark background as well, ideally following the color scheme laid out in the Foundations section of this design system. The social media icons should be adjusted to the necessary icons for one's social media, but should remain within the correct color scheme.

Accessibility

Ensure that the contrast is high enough if not following the provided color scheme in this design system.

Navigation Bar

For Desktop

This is the navigation bar used in the desktop version of the site.


For Desktop [Selected]

This component shows the styling for a the current selected page in the navigation bar.


For Mobile/Tablet

The navigation bar becomes a collapsible hambuger menu button.


Usage & Best Practices

The navigation bar should be placed at the top right of the page. Ensure that the active page is highlighted with the lightest (#f3f3f3) text color.

Accessibility

The navigation bar is made to be responsive using Bootstrap; it should collapse into the hamburger menu at the medium breaking point (for best accessibility on tablet).
